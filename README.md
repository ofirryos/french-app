# 🇫🇷 לומד צרפתית — PWA

## קבצים בתיקייה
- `index.html` — האפליקציה המלאה
- `manifest.json` — הגדרות PWA
- `sw.js` — Service Worker (מאפשר עבודה אופליין)

---

## 🚀 העלאה ל-Netlify (מומלץ - חינם)

1. היכנס ל-[netlify.com](https://netlify.com) וצור חשבון חינם
2. גרור את **כל התיקייה** לאזור ה-Deploy באתר
3. תקבל כתובת URL (לדוגמה: `https://french-abc123.netlify.app`)
4. פתח את הכתובת בדפדפן הפלאפון
5. **iOS Safari**: לחץ כפתור שיתוף ← "הוסף למסך הבית"
6. **Android Chrome**: לחץ תפריט ⋮ ← "התקן אפליקציה"

---

## 🐙 העלאה ל-GitHub Pages (חינם)

1. צור חשבון ב-[github.com](https://github.com)
2. צור Repository חדש
3. העלה את כל הקבצים
4. הפעל GitHub Pages מהגדרות ה-Repository
5. פתח את הכתובת בפלאפון

---

## 📱 שימוש מקומי (ללא שרת)

ניתן לפתוח את `index.html` ישירות בדפדפן, אך:
- Service Worker לא יפעל (דורש HTTPS)
- כל שאר הפיצ'רים יעבדו רגיל
- המילים נשמרות ב-LocalStorage בין רענונים

---

## ✨ פיצ'רים

- 📁 תיקיות מקוננות עם ספירת מילים
- 📝 רשימות מילים עם עריכה ישירה
- 🃏 כרטיסיות עם הפיכה 3D ואנימציה
- 🎮 משחק התאמה צרפתית-עברית
- 🔊 הקראת מילים בצרפתית (TTS)
- 📖 מילון מובנה עם ~100 מילים
- 💾 שמירה אוטומטית ב-LocalStorage
- 📵 עבודה מלאה ללא אינטרנט (לאחר טעינה ראשונה)
